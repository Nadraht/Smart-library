# Smart Library Circulation & Automation System (SLCAS)

## Course
COS 202 – Object-Oriented Programming

## Project Description
This project implements a Smart Library Circulation & Automation System that manages
library items, users, borrowing and returning workflows, searching, sorting, automated
reminders, and reporting using Java, data structures, algorithms, and GUI programming.

## Features
- Add new library items
- Borrow and return items
- Reservation queue system
- Search and sort library items
- Undo last admin operation
- Graphical user interface
- Save and load data from files
- Generate reports

## Technologies Used
- Java (Eclipse Temurin JDK 17)
- Java Swing (GUI)
- Git & GitHub
- VS Code
